# Car Dekho Sales Analysis

## Project Overview

This project analyzes car sales data using SQL to uncover trends in vehicle inventory, fuel type preferences, and yearly sales performance. The analysis provides actionable insights for dealerships and automotive businesses.

**Dataset:** Car Dekho vehicle database  
**Tools:** MySQL  
**Queries:** 25+ SQL queries covering basic to advanced analysis
